Operation of Function:

Operations of functions involve combining two or more functions using basic arithmetic operations: addition, subtraction, multiplication, and division. These operations create new functions by applying the respective operation to the outputs of the original functions

Let's say f(x) = x + 2 and g(x) = x - 3. * Addition: (f + g)(x) = (x + 2) + (x - 3) = 2x - 1 * Subtraction: (f - g)(x) = (x + 2) - (x - 3) = 5 * Multiplication: (f * g)(x) = (x + 2)(x - 3) = x² - x - 6 * Division: (f / g)(x) = (x + 2) / (x - 3), where x ≠ 3 * Composition (f o g)(x): f(g(x)) = f(x - 3) = (x - 3) + 2 = x - 1
